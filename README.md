# publisher


## Quick Installation
**Laravel 5.1:** `composer require tok3/publisher`

#### Service Provider
`Tok3\Publisher\PublisherServiceProvider::class`

**Laravel 5.1**
` 'Publisher' => Tok3\Publisher\PublisherFacade::class,`


And that's it! Start build and publish pages and articles!
